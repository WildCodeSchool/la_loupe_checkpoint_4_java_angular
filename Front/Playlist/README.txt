*******************************Lancer l'application Front*********************************************

Aller dans Front puis dans Playlist.


Une fois dans le dossier Playlist lancer votre terminal depuis ce dossier.

taper la commande "npm install"  ensuite une fois éxécuté tapé la commande "ng serve -o".

ATTENTION le back doit être lancer 

voir le readme.txt du dossier back
