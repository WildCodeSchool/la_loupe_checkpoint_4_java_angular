================================== PlayList Front ==================================

- Sur votre machine dans le répertoire du projet, puis le répertoire front/playliste/ taper "npm install"
- vous pouvez maintenant lancer le projet avec "ng serve" (penser a démarrer le back, voire readme.txt du dossier back )

================================== Deployement =====================================

- taper la commande "ng build --prod"
- copier les fichiers du répertoire build sur votre serveur dans front
