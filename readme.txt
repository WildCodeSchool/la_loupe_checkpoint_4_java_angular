comment utiliser cette app?
___________________________

====================
	FRONT
====================
Tout d'abord rendez-vous dans le dossier front
puis ouvrez un Terminal et tapez la commande suivante:
	$ npm install

pui tout les modules qui sont nessaire au fonctionement de l'app vont se télécharger.
Quand c'est terminer tapez la commande suivante:
	$ sudo ng s -o 

pour lancé l'application.(une fenetre de votre navigateur par defaut s'ouvrira 
	une fois la compilation de l'app terminer)

====================
	BACK
====================
Une fois l'app front lancée laissé votre votre terminal avec la compilation qui tourne, et rendez vous dans le dossier back.

une fois dans le dossier back vous ouvrez un nouveau terminal puis vous tapez la commande suivante:
	$ mvn spring-boot:run

et si tout va bien votre application est prete à etre utiliser
